       - Import `createClientDriveFolder` from `@/lib/google-drive`
       - After the `prisma.$transaction` succeeds (after line 102, before the sendWelcomeEmail call), add Drive folder creation:
         ```
         // Create Google Drive folder (fire and forget - don't block client creation)
         try {
           const driveFolderId = await createClientDriveFolder(companyName, [client ID from transaction])
           await prisma.client.update({
             where: { id: [client ID] },
             data: { driveFolderId }
           })
         } catch (err) {
           console.error('Drive folder creation failed:', err)
           // Don't fail client creation - folder can be created later
         }
         ```
       - To access the client ID from the transaction, change the transaction to return the client: `const { client } = await prisma.$transaction(async (tx) => { ... return { client } })`
       - The transaction currently doesn't return anything. Modify it to return the created client object so we can use `client.id` for the Drive folder creation.

    IMPORTANT: Drive folder creation is OUTSIDE the transaction because it's an external HTTP call. If Drive fails, the client is still created successfully - the admin can retry folder creation later.
