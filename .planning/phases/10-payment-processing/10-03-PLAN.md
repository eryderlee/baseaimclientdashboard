- "Pay Now" button (shown when status is SENT or OVERDUE):
  - On click: fetch `/api/invoices/${invoiceId}/urls` to get `hosted_invoice_url`

  - Show loading spinner while fetching
  - If no URL, show toast error "PDF not available"
- Use shadcn Button components, Download icon from lucide-react

**ManageBillingButton component:**
- Standalone button: "Manage Billing"
- On click: call `createPortalSession` server action
- If success, `window.location.href = url` to redirect to Stripe Customer Portal
- If error, show toast with error message
- Use useTransition for loading state
- Import createPortalSession from '@/app/actions/billing'
- Style: outline variant, CreditCard icon

Both components should handle loading states gracefully (disabled button + spinner).
