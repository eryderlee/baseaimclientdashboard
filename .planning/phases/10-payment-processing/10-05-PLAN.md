  <files>components/admin/subscription-manager.tsx</files>
  <action>
Create `components/admin/subscription-manager.tsx` as a client component ('use client').

Props:
```typescript
interface SubscriptionManagerProps {
  clientId: string
  subscription: {
    id: string
    status: string
    currentPeriodEnd: string | null  // ISO string (serialized from server)
    stripePriceId: string | null
    stripeSubscriptionId: string | null
  } | null
}
```

The component has two states:

---

**State A — No active subscription** (subscription is null, or status is 'inactive'):

Show a Card with:
- Title: "Monthly Retainer" with a `RefreshCw` icon from lucide-react
- Description: "Start automatic monthly billing for this client"
- A simple form with:
  - Monthly Amount field: number input, step="0.01", min="0.01", placeholder="0.00", labelled "Monthly Amount (AUD)"
  - Description field: text input, defaultValue="Monthly Retainer", labelled "Description"
  - Submit button: "Start Retainer" with loading state via useTransition
- On submit:
  ```typescript
  const formData = new FormData()
  formData.set('clientId', clientId)
  formData.set('monthlyAmount', String(monthlyAmount))
  formData.set('description', description)
  formData.set('currency', 'aud')
  const result = await startSubscription(formData)
  ```
- On success: `toast.success('Monthly retainer started — first invoice sent to client')`; page will revalidate and show State B
- On error: `toast.error(result.error || 'Failed to start retainer')`

Use local state (useState) for the form inputs — no react-hook-form needed for this simple form.

---

**State B — Active or cancelling subscription** (subscription exists and status is 'active' or 'cancelling'):

Show a Card with:
- Title: "Monthly Retainer" with a `RefreshCw` icon
- Badge: status === 'active' ? green "Active" : amber "Cancelling"
- If status is 'active': show next billing date from `currentPeriodEnd` formatted as `new Intl.DateTimeFormat('en-AU', { day: 'numeric', month: 'long', year: 'numeric' }).format(new Date(subscription.currentPeriodEnd))`
- If status is 'cancelling': show "Subscription ends [date]" message
- Cancel button (only when status is 'active'):
  - Label: "Cancel Retainer"
  - variant="outline" with `XCircle` icon
  - On click: show a confirm dialog — use `window.confirm('Cancel this client\'s monthly retainer? They will keep access until the end of the current billing period.')`
  - If confirmed: call cancelSubscription via useTransition
  ```typescript
  const formData = new FormData()
  formData.set('clientId', clientId)
  const result = await cancelSubscription(formData)
  ```
  - On success: `toast.success('Retainer cancelled — active until end of billing period')`
  - On error: `toast.error(result.error || 'Failed to cancel')`

Import startSubscription and cancelSubscription from '@/app/actions/billing'.
