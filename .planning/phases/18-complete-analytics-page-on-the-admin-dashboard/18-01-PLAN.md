  <files>lib/dal.ts</files>
  <action>
Add two new DAL functions after the existing `getAdminFbAggregation` function in lib/dal.ts.

**getAdminFbPerClient():**
- verifySession() BEFORE unstable_cache boundary (ADMIN role only)
- Get settings via getSettings() — return empty object `{}` if no facebookAccessToken
- Query all clients where adAccountId is not null: `prisma.client.findMany({ where: { adAccountId: { not: null } }, select: { id: true, adAccountId: true } })`
- Return `{}` if no clients found
- Inside unstable_cache (key: `['admin-fb-per-client-v1']`, revalidate: 21600):
  - Use Promise.allSettled to call `fetchFacebookInsights(c.adAccountId!, 'last_30d', accessToken)` for each client
  - Build a Record<string, { spend: number; leads: number }> keyed by client.id
  - For fulfilled results: spend = parseFloat(result.value.spend || '0'), leads = getActionValue(result.value.actions, 'lead')
  - Skip rejected results (do not include client in map)
- Return type: `Record<string, { spend: number; leads: number }>`
- Wrap in React cache()

**getAdminFbDailyAggregation():**
- verifySession() BEFORE unstable_cache boundary (ADMIN role only)
- Get settings via getSettings() — return `[]` if no facebookAccessToken
- Query all clients where adAccountId is not null (same query pattern)
- Return `[]` if no clients found
- Inside unstable_cache (key: `['admin-fb-daily-agg-v1']`, revalidate: 21600):
  - Use Promise.allSettled to call `fetchFacebookDailyInsights(c.adAccountId!, accessToken, 'last_30d')` for each client
  - Build a Map<string, { spend: number; leads: number }> keyed by date string (date_start from FbDailyInsight)
  - For each fulfilled result (array of FbDailyInsight[]):
    - For each day: spend += parseFloat(day.spend || '0'), leads += getActionValue(day.actions, 'lead')
  - Convert Map to sorted array: `Array.from(map.entries()).sort(([a], [b]) => a.localeCompare(b)).map(([date, vals]) => ({ date, spend: vals.spend, leads: vals.leads }))`
- Return type: `Array<{ date: string; spend: number; leads: number }>`
- Wrap in React cache()

Both functions follow the exact same pattern as getAdminFbAggregation (verifySession outside cache, Promise.allSettled, parseFloat on FB values, getSettings for token).
