**Seed logic:**

The `seed()` function:

1. Hash the demo admin password once: `const hashedPwd = await bcrypt.hash(DEMO_ADMIN_PASSWORD, 10)`

2. Upsert demo admin user:
```typescript
await prisma.user.upsert({
  where: { email: DEMO_ADMIN_EMAIL },
  update: { name: 'Zara Khan', role: 'ADMIN' },
  create: { email: DEMO_ADMIN_EMAIL, name: 'Zara Khan', password: hashedPwd, role: 'ADMIN' },
})
```

3. For each of the 5 client profiles, in a loop:
   a. Upsert User by email (create with CLIENT role, hashed password, name)
   b. Upsert Client by demoStableId:
      - `where: { demoStableId: profile.stableId }`
      - `update: { companyName, industry, onboardingStep, adAccountId, leadsChartEnabled, isDemo: true }`
      - `create: { userId: user.id, companyName, industry, onboardingStep, adAccountId, leadsChartEnabled, isDemo: true, demoStableId: profile.stableId }`
   c. Delete existing milestones for this client: `prisma.milestone.deleteMany({ where: { clientId: client.id } })`
   d. Create milestones via `prisma.milestone.createMany({ data: [...] })`
   e. Delete existing invoices: `prisma.invoice.deleteMany({ where: { clientId: client.id } })`
   f. Create invoices via `prisma.invoice.createMany({ data: [...] })` (only for post-setup clients)
   g. Delete existing documents: `prisma.document.deleteMany({ where: { clientId: client.id } })`
   h. Create documents via `prisma.document.createMany({ data: [...] })` (only for post-setup clients)

   For documents, use `fileUrl: '#demo'` as a placeholder URL — the URL is only accessed on download click, acceptable for demo purposes. Set `uploadedBy: 'BaseAim Team'`, `status: 'APPROVED'`.

   For invoices, set `items` as a JSON array: `[{ description: '...', quantity: 1, unitPrice: amount }]`.

**Clean logic:**

The `clean()` function:
1. Delete all demo clients (cascade handles milestones, invoices, documents): `await prisma.client.deleteMany({ where: { isDemo: true } })`
2. Delete demo client user accounts: `await prisma.user.deleteMany({ where: { email: { in: [all 5 demo client emails] } } })`
3. Delete demo admin: `await prisma.user.deleteMany({ where: { email: DEMO_ADMIN_EMAIL } })`
4. Print: `Demo data removed. (1 admin + 5 clients and all associated records)`

**Main:**
```typescript
async function main() {
  if (CLEAN) { await clean(); return }
  await seed()
}

main()
  .catch(console.error)
  .finally(() => prisma.$disconnect())
```
